## Tone guidelines

If you adapt or translate the copy, keep these conventions:

- **No emojis.** Product is technical; the audience respects restraint.
- **Concrete over abstract.** Prefer "your NAS at `nas.example.com`" over "self-hosted infrastructure".
- **Honest about scope.** This is homelab / prosumer / small-team. Do not claim enterprise features it doesn't have.
- **Respect the comparison targets.** Tailscale, Cloudflare Tunnel, and DynDNS are legitimate tools; the point is "different trade-offs", not "they're bad".
- **Own the freedom angle.** The differentiator is ownership and self-hosting, not raw feature count.
